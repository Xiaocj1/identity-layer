# Contributing to the Send in Peace Protocol

## What this repository IS
- A protocol proposal and governance experiment.
- The canonical source for the `GENESIS.md` principles.
- **The trigger box for the autonomous stewardship transition** (see `GOVERNANCE_TRANSITION.md`).

## What this repository is NOT
- A codebase for a specific implementation.
- A general discussion forum (use **GitHub Discussions**).

## How to contribute to THIS REPO (before the transition)
We only accept minor fixes to:
- `README.md` (for clarity)
- `GENESIS.md` (for typographical errors only)
- Files in the `Whitepaper/` directory (for translations or typos)
